# 🗂 Async File Sorter

Цей проєкт містить Python-скрипт для асинхронного сортування файлів за їхніми розширеннями. Скрипт рекурсивно читає вихідну папку та ефективно розподіляє файли по відповідних підпапках у цільовій директорії.

---

## 🛠 Технології та Особливості
- **Асинхронність:** Використовуються `asyncio` та `aiofiles` для швидкого паралельного читання та копіювання файлів.
- **Безпека:** Файли *копіюються*, а не переміщуються. Оригінали завжди залишаються недоторканими.
- **Обробка виключень:** Усі помилки доступу або читання перехоплюються та логуються безпечним способом без припинення роботи скрипта.
- **Зручний CLI:** Приймає аргументи з командного рядка для гнучкого налаштування джерела та призначення.

---

## 🚀 Встановлення та Запуск

### 1. Підготовка середовища
Оскільки скрипт використовує зовнішню бібліотеку `aiofiles`, її потрібно встановити:

```bash
pip install aiofiles
```

### 2. Використання
Синтаксис запуску:
```bash
python sort_files.py <вихідна_папка> [цільова_папка]
```

- `<вихідна_папка>` — обов'язковий аргумент, шлях до директорії, де лежать файли для сортування.
- `[цільова_папка]` — необов'язковий аргумент, шлях куди будуть збережені відсортовані файли (За замовчуванням: `dist`).

**Приклад запуску:**
```bash
python sort_files.py my_downloads sorted_files
```

---

## 📂 Приклад результату роботи

Якщо у вас у вихідній папці були файли `document.txt`, `image.png`, `script.py` і файл без розширення `unknown_file`, то після запуску ви отримаєте наступну структуру в цільовій папці:

```text
sorted_files/
├── txt/
│   └── document.txt
├── png/
│   └── image.png
├── py/
│   └── script.py
└── no_extension/
    └── unknown_file
```
